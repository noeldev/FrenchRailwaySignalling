Ce template est utilisé pour documenter les plaques qui identifient les principaux signaux lumineux ferroviaires français.

== Exemples ==
 <code><nowiki>
{{Template:FR:RailwaySignalPlate
|category=distant
|signal=A
|plate=A
|image_file=Plaque A.svg
|description=Cette plaque identifie un avertissement de BAPR.
}}

{{Template:FR:RailwaySignalPlate
|signal=CARRE
|plate=PR
|caption=PR vers xxx
|image_file=Plaque PR vers xxx.svg
|description=Cette plaque identifie le mode de cantonnement '''BAPR''' pour le ou les cantons en aval du signal.
}}
</nowiki></code>
